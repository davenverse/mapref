    Quick Start

    To use mapref in an existing SBT project with Scala 2.11 or a later version, add the following dependencies to your build.sbt depending on your needs:

    libraryDependencies ++= Seq(
  "io.chrisdavenport" %% "mapref" % "<version>"
)
